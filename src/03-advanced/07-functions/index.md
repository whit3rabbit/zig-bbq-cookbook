# Chapter 7: Functions

Master Zig's function patterns and unique features.

## Topics

- Variable argument functions (using slices or tuples)
- Keyword-only arguments (configuration structs)
- Function metadata and reflection
- Returning multiple values (structs/tuples)
- Default arguments (struct parameters with default fields)
- Anonymous/inline functions (anonymous structs with `call` methods)
- Capturing variables (manual closure creation)
- Partial application (N-argument to fewer arguments)
- Function pointers and contexts
- Carrying extra state with callbacks (`*anyopaque` contexts)
- Inlining callback functions
