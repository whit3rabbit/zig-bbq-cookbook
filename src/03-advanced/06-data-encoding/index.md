# Chapter 6: Data Encoding and Processing

Learn to process various data formats in Zig.

## Topics

- Reading and writing CSV data
- JSON processing (deep dive into `std.json`, streaming vs. DOM)
- Simple XML parsing (3rd party or libxml2)
- Incremental huge XML file parsing
- Dictionary to XML conversion
- Relational database interaction (SQLite from Zig)
- Hexadecimal encoding/decoding (`std.fmt.fmtSliceHex`)
- Base64 encoding/decoding (`std.base64`)
- Binary arrays of structures (`std.mem.bytesAsSlice`)
