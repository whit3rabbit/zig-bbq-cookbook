# Chapter 17: Advanced Comptime Metaprogramming

Unlock the full power of Zig's compile-time execution.

## Topics

- Creating generic data structures (functions returning Types)
- Compile-time string processing
- Compile-time reflection (iterating struct fields with `@typeInfo`)
- Validating data at compile time (`@compileError`)
- Unrolling loops (`inline for`, `inline while`)
- Creating Type IDs or perfect hashes at compile time
- Implementing duck-typing via comptime interface checks (`hasDecl`)
