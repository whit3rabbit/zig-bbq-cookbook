# Phase 1: Foundation & Philosophy

This phase establishes the core principles and patterns used throughout the cookbook.

## Topics Covered

- **Idiomatic Zig** - Writing Zig code the right way
- **Error Handling Patterns** - When to use `try`, `catch`, or `if (res) |val|`
- **Testing Strategy** - Using `std.testing` within same-file `test` blocks

## Learning Objectives

- Understand standard allocator usage patterns (e.g., passing allocators as first argument to init functions)
- Master Zig's error handling philosophy
- Establish testing practices for all recipes

See individual sections for detailed recipes.
