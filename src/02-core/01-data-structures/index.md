# Chapter 1: Data Structures & Algorithms

Learn to work with Zig's standard library containers and manual memory management.

## Topics

- Unpacking/Destructuring (Tuples and Arrays)
- Working with slices (`[]T`)
- Ring Buffers and `std.fifo`
- Sorting and finding largest/smallest items
- Priority Queues (`std.PriorityQueue`)
- HashMaps and ArrayHashMaps
- Set operations and deduplication
- Grouping and filtering sequences
- Custom comparators and sorting
