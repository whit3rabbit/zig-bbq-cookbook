# Chapter 3: Numbers, Dates, and Times

Work with Zig's precise numeric types and time handling.

## Topics

- Rounding values (`@round`, `@floor`, `@ceil`)
- Accurate decimal calculations (fixed-point arithmetic)
- Formatting numbers (`std.fmt` format strings)
- Binary, octal, and hexadecimal integers
- Packing/unpacking integers from bytes (`std.mem.readInt`, `writeInt`)
- Complex-valued math (`std.math.Complex`)
- Infinity and NaNs
- Fractions and rationals
- Large numerical arrays (`@Vector` for SIMD)
- Matrix and linear algebra
- Random number generation (`std.rand.DefaultPrng`)
- Time conversions (`std.time`)
