# Phase 4: Specialized Topics

This phase covers domain-specific programming areas where Zig excels.

## Chapters

11. **Network and Web Programming** - `std.net` and `std.http`
12. **Concurrency** - `std.Thread`, Mutexes, and potentially `async`
13. **Utility Scripting and System Administration** - Replacing Python glue scripts with performant Zig
14. **Testing, Debugging, and Exceptions** - `std.testing` and Error Sets
15. **C Interoperability** - Zig calling C, and C calling Zig

## Focus Areas

Learn to build production-ready systems that interact with networks, multiple threads, external processes, and C libraries.
