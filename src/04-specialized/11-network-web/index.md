# Chapter 11: Network and Web Programming

Build networked applications with Zig's standard library.

## Topics

- HTTP client operations (`std.http.Client`)
- TCP server creation (`std.net.StreamServer`)
- UDP servers
- IP address ranges from CIDR
- REST-based interfaces
- XML-RPC implementation
- Inter-process communication (IPC with pipes/sockets)
- Remote procedure calls
- Client authentication
- Adding SSL (integrating BearSSL or OpenSSL)
- Passing socket file descriptors between processes
- Event-driven I/O (`std.event.Loop`, `epoll`/`kqueue`)
