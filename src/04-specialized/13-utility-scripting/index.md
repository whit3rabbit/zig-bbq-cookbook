# Chapter 13: Utility Scripting and System Administration

Build system utilities and automation tools in Zig.

## Topics

- Script input via redirection, pipes, or files
- Terminating programs with error messages
- Command-line option parsing
- Runtime password prompts
- Getting terminal size
- Executing external commands (`std.process.Child`)
- Copying/moving files and directories
- Creating and unpacking archives (zlib/libtar)
- Finding files by name (recursive directory walking)
- Reading configuration files (INI or JSON parsing)
- Logging for scripts (`std.log`)
- Logging for libraries (custom log scopes)
- Stopwatch timers (`std.time.Timer`)
- Memory and CPU usage limits
- Launching web browsers
