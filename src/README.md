# The Zig BBQ Cookbook

Welcome to the Zig BBQ Cookbook - your comprehensive guide to Zig programming through practical, tested recipes.

## About This Book

This cookbook is organized into 5 phases covering 20 chapters with ~200 recipes:

- **Phase 1: Foundation & Philosophy** - Idiomatic Zig patterns
- **Phase 2: Core Recipes** - Essential Zig programming techniques
- **Phase 3: Advanced Topics** - Deep dives into Zig's powerful features
- **Phase 4: Specialized Topics** - Network, concurrency, and C interop
- **Phase 5: Zig-Specific Paradigms** - Unique Zig capabilities

All code samples are tested with **Zig v0.15.2**.
